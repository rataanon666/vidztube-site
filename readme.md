# VIDZTUBE

**VIDZTUBE** is a privacy-focused video streaming platform designed to operate through the Tor network.

The project aims to provide anonymous access to video content while prioritizing privacy and censorship resistance.

---

## Overview

VIDZTUBE is an experimental platform that explores anonymous video distribution using privacy-preserving technologies.

The platform is designed with the following principles:

* Privacy first
* Anonymous access
* Minimal tracking
* Decentralized vision

---

## Accessing VIDZTUBE

VIDZTUBE is accessible through the Tor network.

1. Download the Tor Browser from the official website
2. Install and launch the browser
3. Open the VIDZTUBE onion address

Official Tor Browser website:

https://www.torproject.org

* Anonymous video streaming
